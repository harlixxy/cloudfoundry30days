這是一本關於 Cloud Foundry 開放源碼 PaaS 的電子書，我們使用 GitHub 維護電子書內容，並交由 ContPub（Continuous Publishing）系統自動線上發佈。本書提供 PDF、EPUB、MOBI 及 HTML 等格式，您除了可以在網站檢視本書所有內容，也可以將電子書下載至閱讀器保存。
